# Minimalist Technical Blog

A single-file technical blog built with HTML, CSS, and JavaScript.

## Features

- Minimalist responsive layout
- About Me section
- Technical Log section
- Contact section
- English/Japanese language toggle
- Single-file structure using `index.html`

## Purpose

This project is a personal technical blog for recording development progress, debugging notes, project lessons, and technical Japanese practice.

## Tech Stack

- HTML
- CSS
- JavaScript
