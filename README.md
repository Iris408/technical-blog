# ミニマル技術ブログ

HTML、CSS、JavaScript で作成した single-file の技術ブログです。

## 機能

* ミニマルでレスポンシブなレイアウト
* 自己紹介セクション
* 技術ログセクション
* 連絡先セクション
* 日本語 / 英語の言語切り替え
* index.html のみで構成された single-file 構造

## 目的

このプロジェクトは、開発の進捗、デバッグ記録、プロジェクトで学んだこと、技術日本語の練習を記録するための個人技術ブログです。

## 使用技術

* HTML
* CSS
* JavaScript
